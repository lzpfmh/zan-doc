目录结构
========

测试文件位于zan/test文件夹，其下的目录结构为

.. code:: php

    test
    ├── bootstrap.php
    ├── Foundation
    ├── Network
    ├── resource
    ├── Sdk
    ├── Store
    ├── TestCase.php
    ├── Testing
    ├── Utilities
    └── vendor

-  测试文件目录与src文件目录基本一致，resource为测试的配置文件夹，vendor为测试依赖的其他文件。
-  可以在测试文件夹中找到对应接口的用法。
